# 💰 個人資產管理 App

一個純前端單頁應用，用於追蹤個人資產、收入、支出、投資組合與目標。

## 功能
- 📊 總覽：資產淨額趨勢、資產配置、收支圖表
- 💵 收入儲蓄：月收入、儲蓄率、銀行帳戶餘額
- 💳 支出分析：分類支出統計與趨勢
- 📈 投資組合：持倉明細 CRUD、對帳單更新
- 🌸 股利追蹤：股利紀錄與發放時程
- 🎯 目標追蹤：資產目標進度、個人目標管理
- 📅 年度總結：年度財務比較
- 📤📥 匯出 / 匯入：跨裝置同步資料（JSON 備份）

## 技術
- 純 HTML + CSS + JavaScript（單檔，無需框架）
- Chart.js 4.4.0（圖表）
- localStorage 本地儲存

## 部署
此 repo 已啟用 GitHub Pages，直接訪問即可使用。

> **注意**：資料存在瀏覽器的 localStorage，不同裝置請使用「📤 匯出」備份，再到新裝置「📥 匯入」。
